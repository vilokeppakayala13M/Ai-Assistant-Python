AI Assistant Python Project

A simple, customizable AI assistant built with Python, leveraging APIs like OpenAI or local models for tasks such as chat, reminders, and web queries. Perfect for your first mobile game dev side project or portfolio showcase.

Features
a. Interactive command-line or GUI chat interface.
b. Supports multiple AI backends (e.g., GPT, Ollama).
c. Handles user queries with context memory.
d. Extensible for voice input/output or integrations.

Tech Stack
i.Python 3.8+
ii.openai or ollama for AI core.
iii tkinter for optional GUI.
iv dotenv for API keys.

Quick Setup
1. Clone the repo: git clone <your-repo-url>
2. Install dependencies: pip install -r requirements.txt
3. Set up .env: Add OPENAI_API_KEY=your_key (or use local model).
4. Run: python main.py
